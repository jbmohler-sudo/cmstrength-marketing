1. The Visual Identity (The "Rugged Pro" Spec)
Background: True Black (#000000).

Primary Accent: Cyan Glow (#0ea5e9 with shadow/blur effects).

Typography: Bold, italicized headers for authority; Inter/Sans-serif for utility.

UI Philosophy: Glassmorphism (semi-transparent, blurred backgrounds) and high-contrast borders. No generic white-space.

2. The Core Engines (The "Meat & Potatoes")
Macrocycle Engine: 13-week wave architecture (Foundation → Strength → Peak).

Load Engine: Percentage-based, reactive weights using the Training Corridor (MEV to MRV).

Readiness Engine: Daily interview data (Stress, Sleep, Fatigue) + Joint-specific pivots.

Compliance Logic: Calculated as Completed Sessions / Sessions Scheduled Until Today.

3. Frontend Architecture (The HubSpot Pivot)
Hero Section: Static HUD featuring the Macrocycle Wave centerpiece and telemetry flankers.

Feature Sections: Triple-Chapter Sticky Scroll (Adaptation, Sovereignty, The Engines).

Support: Elegant Accordion FAQ (Split-pane layout).

4. Developer Guardrails
Deployment: Vercel (main branch is production).

File Structure: Methodology vault in /methodology/, assets in /public/assets/, internal strategy in /docs/ai-strategy/.

The "No-BS" Rule: Prioritize functional data over marketing fluff. If a feature doesn't help the lifter hit a PR, it's secondary.
